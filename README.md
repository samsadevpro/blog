# Mon Blog star wars

## Description

Bienvenue sur mon bolg ! C'est un site simple pour présenter des personnages de star wars.

## Installation

Clonez le projet et ouvrez le fichier `index.html` dans un navigateur moderne pour voir le site.

## Utilisation

1. Clonez ce projet sur votre machine locale.
2. Ouvrez `index.html` dans un navigateur pour voir le site.
